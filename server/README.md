# Server Integrations

These helpers are optional server-side examples for Turnstile and Resend.
They are not used by the client app directly.

## Turnstile verification
- File: `server/turnstile/verify-turnstile.js`
- Env: `TURNSTILE_SECRET_KEY`

## Resend email
- File: `server/resend/send-email.js`
- Env: `RESEND_API_KEY`, `RESEND_FROM`
